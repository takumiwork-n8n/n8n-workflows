## 🎮 How to Play (Daily Routine)
このロードマップは、AIエージェントと対話しながら進める「ゲームの攻略本」です。

### 1. ログイン (Start)
- 朝起きたら、Agentに一言: **「今日のタスクは？」**
- Agent (Mentor) がスケジュールを確認し、その日の「写経課題」や「教材」へのリンクを提示します。

### 2. クエスト攻略 (Execute)
- 提示されたテンプレートを写経（作成）します。
- 詰まったら: **「ここエラー出るんだけど教えて」** (Debug Dojoモード発動)
- 終わったら: **「写経できた。次は？」** (Custom Challenge発動)

### 3. セーブ (Save)
- ワークフローができたら: **「GitHubに保存して」**
- Agent (Expert) が自動でJSON解析・README作成・Pushまで行います。

### 4. ログアウト (End)
- 1日の終わりに: **「今日のログ書いて」**
- 学びをObsidianに記録し、進捗率を更新して終了です。

## 📈 日々の学習サイクル（写経・カスタム）
1. **Challenge**: メンターが世界中の優秀なテンプレートから1つを選定・提示。
2. **Shakyo**: 提示されたテンプレートを何も見ずに再現できるまで作り込む。
3. **Custom**: そのテンプレートに「自分だけの機能」を1つ追加（カスタマイズ）。
4. **Log**: 構築したJSONと気づきをObsidianに記録し、メンターへ報告。
