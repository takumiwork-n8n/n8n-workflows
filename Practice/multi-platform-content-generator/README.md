# Multi-Platform Content Generator

SNS（X, Threads, Instagram, note）向けのコンテンツ案を自動生成するワークフローです。

## アップデート情報
- プロンプトの微調整を行いました。
- フォルダ構造の整理に対応しました。

## 主な機能
- スケジュール実行（毎日9時）
- ペルソナ設定に基づいたコンテンツ生成
- 4つのプラットフォームへの同時展開向けの投稿アイデアを自動生成するツールです。

## 概要
- **自動生成**: X, Threads, Instagram, note用の投稿案を一括生成。
- **カスタマイズ**: ペルソナ設定ノードでトーン＆マナーを調整可能。
