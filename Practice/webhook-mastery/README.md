# Webhook Mastery: Inspirational Quotes API

## 概要
このワークフローは、n8nをシンプルかつ強力な**APIサーバー**として利用するための学習用プロジェクトです。
ユーザーからのリクエストをWebhookで受け取り、外部API（ZenQuotes）から名言を取得し、整形したJSONレスポンスを返却します。

## 学習のゴール (Learning Outcomes)
- **Webhook Trigger**: `GET` メソッドによるリクエスト受信。
- **HTTP Request**: 外部APIへのアクセスとデータ取得。
- **Data Transformation**: `Set` ノードを使ったデータの抽出と文字列整形。
- **Webhook Response**: `Respond to Webhook` ノードを利用した、明示的なJSONレスポンスの返却。

## 構成
1. **Webhook (Trigger)**: `/webhook/test` などのパスでリクエストを待機。
2. **HTTP Request**: ZenQuotes API (`https://zenquotes.io/api/random`) をコール。
3. **Set (Edit Fields)**: 取得した名言 (`q`) と著者 (`a`) を連結し、`quotes` 変数に格納。
   - 式: `{{ $json.q }} - {{ $json.a }}`
4. **Respond to Webhook**: クライアントに `Content-Type: application/json` で結果を返却。

## 使い方 (How to use)
1. ワークフローをインポートする。
2. Webhookノードを開き、`Test URL` をコピーする。
3. ブラウザのアドレスバーにURLを貼り付けてアクセスする。
4. 画面に `{"quotes": "名言 - 著者"}` というJSONが表示されれば成功。
